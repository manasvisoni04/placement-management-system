# 🎓 Placement Management System

A full-stack web application designed to streamline campus placement processes with role-based dashboards for Admin and Students.

---

## 🚀 Features

* 🔐 Authentication using Firebase
* 👨‍💼 Admin Dashboard

  * Add and manage placement drives
  * View student queries and applications
* 🎓 Student Dashboard

  * Browse placement opportunities
  * Apply for jobs
  * Raise queries
* 📊 Real-time data using Firestore

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite), Tailwind CSS
* **Backend:** Firebase
* **Database:** Firestore
* **Authentication:** Firebase Auth
* **Hosting:** Firebase Hosting

---

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/manasvisoni04/placement-management-system.git
```

2. Navigate to project:

```bash
cd placement-management-system/client
```

3. Install dependencies:

```bash
npm install
```

4. Create `.env` file using `.env.example`

5. Run the app:

```bash
npm run dev
```
